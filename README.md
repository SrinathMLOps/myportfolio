# Portfolio Website

A modern, responsive portfolio website with a dark theme.

## Setup Instructions

1. **Customize Your Content**
   - Open `index.html` and replace placeholder text with your information:
     - Your name
     - Job title
     - Location
     - Email and phone
     - About me description
     - Services/skills
     - Testimonials
     - Client information

2. **Add Your Images**
   - `profile.jpg` - Your profile picture (recommended: 300x300px)
   - `testimonial1.jpg`, `testimonial2.jpg` - Testimonial photos (recommended: 200x200px)
   - `client1.png`, `client2.png`, etc. - Client logos (recommended: transparent PNG)

3. **Test Locally**
   - Simply open `index.html` in your web browser
   - No server required for basic testing

## Hosting Options

### Option 1: GitHub Pages (Free)
1. Create a GitHub account at https://github.com
2. Create a new repository named `yourusername.github.io`
3. Upload all files (index.html, styles.css, script.js, images)
4. Go to Settings > Pages
5. Select main branch as source
6. Your site will be live at `https://yourusername.github.io`

### Option 2: Netlify (Free)
1. Create account at https://netlify.com
2. Drag and drop your folder to Netlify
3. Get instant custom domain
4. Automatic HTTPS included

### Option 3: Vercel (Free)
1. Create account at https://vercel.com
2. Import your project
3. Deploy with one click
4. Custom domain available

### Option 4: Traditional Hosting
- Use services like Hostinger, Bluehost, or GoDaddy
- Upload files via FTP
- Point your domain to the hosting

## Customization Tips

- Colors: Edit the CSS color values in `styles.css`
- Layout: Modify grid columns in `.services-grid` and `.clients-grid`
- Fonts: Add Google Fonts link in HTML head section
- Sections: Add more sections by copying existing section structure
